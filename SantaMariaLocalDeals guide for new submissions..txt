## ğŸ’¡ Quick Tips

### To Update Multiple Deals at Once:
- Edit `deals.json` with all changes
- Upload once
- All deals update simultaneously

### To Feature Urgent Deals:
- Put them at the top of the deals array in `deals.json`
- They'll appear first on the website

### To Remove Expired Deals:
- Set `"active": false` instead of deleting
- Keep the data for records

### Backup Your Data:
- Download `deals.json` regularly
- Keep a copy on your computer

## ğŸ†˜ Troubleshooting

### Forms Not Working?
- Make sure `data-netlify="true"` is in the form tag
- Check Netlify Forms dashboard for errors
- Verify form name matches in Netlify

### Deals Not Showing?
- Check `deals.json` for syntax errors (use jsonlint.com)
- Verify file is named exactly `deals.json`
- Check browser console for errors (F12 key)

### Need Help?
- Netlify Support: support.netlify.com
- JSON Validator: jsonlint.com
- Test your site: Use different browsers and devices
